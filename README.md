# MistiGameProject
SE 116 // Advanced Pişti Game Project

Proje Hedefi
Projemiz, ileri seviye bir pişti oyunu yapmak ve oyunculara daha iyi bir deneyim sunmaktır. 
Bu hedefe ulaşmak için, pişti oyununun standart kurallarına ek olarak birçok özellik ekleyeceğiz.
Bunlar arasında, yeni kart desteleri, zorluk seviyeleri, ses efektleri ve oyun içi ipuçları yer alabilir.
Ayrıca, oyuncuların oyun deneyimlerini geliştirmek için farklı oyun modları da ekleyebiliriz.
